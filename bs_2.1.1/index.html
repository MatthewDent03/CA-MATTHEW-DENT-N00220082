Grids

To use grids - we need to:

  1. Create A Row!
    This is attached to a DIV. It's contents will flex-i.e. we use it to layout elements horizontally
  2. Put Columns Inside Our Row

    Our columns must add up to 12 since the BS grid is 12 columns wide.

  3. Make Different Values For Our Grids

    If we want to change layouts of our grids depending on our viewports, we can do this by referencing the correct classes!
